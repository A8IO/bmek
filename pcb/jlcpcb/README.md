# Ordering BMEK PCB at JLCPCB

![PCB](https://i.imgur.com/Gihmnn3.png)

Go to [jlcpcb.com](https://jlcpcb.com) and get a quotation. Use the .zip for uploading gerber files. For assembly, use BOM.csv and CPL.csv.
